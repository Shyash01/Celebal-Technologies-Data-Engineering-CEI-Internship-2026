# Week 1 - Basic Data Exploration and Cleaning using Pandas

## Objective
Perform data exploration and cleaning operations using Python Pandas library on an e-commerce shopping dataset.

## Features
- Data loading using Pandas
- Dataset exploration
- Missing value handling
- Duplicate record verification
- Price data cleaning
- Filtering and column selection
- Derived column creation
- Data visualization
- Export cleaned dataset

## Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Output Files
- Jupyter Notebook (.ipynb)
- Cleaned CSV dataset
- Documentation and summary
