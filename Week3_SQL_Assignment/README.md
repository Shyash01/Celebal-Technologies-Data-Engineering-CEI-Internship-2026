# Celebal Summer Internship 2026 - Week 3 SQL Assignment

## Objective

The objective of this assignment is to analyze sales data from the Superstore dataset using SQL concepts such as Subqueries, Common Table Expressions (CTEs), and Window Functions in PostgreSQL.
